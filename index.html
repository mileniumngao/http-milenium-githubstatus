How to Host Your Portfolio Website

GitHub Pages Easiest

  1. Create a GitHub account at github.com
  2. Create a new repository named yourusername.github.io
  3. Upload your HTML file (name it index.html)
  4. Go to Repository Settings → Pages → Select branch
  5. Your site will be live at: https://yourusername.github.io

Netlify Recommended

  1. Sign up at netlify.com
  2. Drag & drop your project folder
  3. Netlify will automatically deploy your site
  4. Get a free subdomain like your-site.netlify.app
  5. Connect custom domain later if needed

Vercel

  1. Create account at vercel.com
  2. Import your GitHub/GitLab repository
  3. Vercel will detect it's a static site
  4. Deploy with default settings
  5. Get a URL like your-site.vercel.app

Cloudflare Pages

  1. Sign up at pages.cloudflare.com
  2. Connect your Git repository
  3. Set build command to none (static site)
  4. Set output directory to root
  5. Deploy and get a *.pages.dev URL
